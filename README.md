# SDL-BlockBreaker
a game about block breaker by SDL/C++

这是用C++和SDL图形库写的美女打砖块游戏


大一的时候写的，代码写的比较糟糕，就当作一个SDL图形库的使用Demo吧。

注意：只支持1080P分辨率,而且背景音乐使用了wav，导致体积比较大，事实上这个SDL是可以用ogg音乐的。
